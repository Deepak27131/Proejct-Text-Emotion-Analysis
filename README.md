# NLP Text Emotion Analysis using AI

## 📌 Project Overview
This project was developed as part of the **Artificial Intelligence Virtual Internship with IBM**. The goal of this project is to analyze human emotions from text using **Natural Language Processing (NLP)** and Artificial Intelligence techniques.

The system processes textual input and predicts the emotional tone behind the text such as happiness, sadness, anger, or neutral sentiment.

## 🚀 Key Features
- Text Emotion Detection using NLP
- AI-based text analysis
- Real-time emotion prediction from text input
- Simple and easy-to-understand implementation

## 🧠 Technologies Used
- Python
- Natural Language Processing (NLP)
- Artificial Intelligence
- Machine Learning
- Jupyter Notebook / Python Scripts

## ⚙️ How It Works
1. User provides a text input.
2. The text is processed using NLP techniques.
3. The AI model analyzes the text patterns.
4. The system predicts the emotion present in the text.
